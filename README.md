# Hand Gesture Controlled Robot

This project demonstrates a computer vision based robotic system that controls robot movements using hand gestures.

The system uses a webcam to detect hand gestures and sends commands to a robot in real time.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Arduino
- CVZone

## Features
- Real-time hand gesture detection
- Robot movement control using gestures
- Computer vision based interaction
- Human-robot interaction without physical controllers

## Applications
- Human Robot Interaction
- Assistive Robotics
- Smart Automation
- Contactless Robot Control

## How It Works
1. Webcam captures hand gestures.
2. MediaPipe detects hand landmarks.
3. Gesture is recognized using Python logic.
4. Command is sent to Arduino.
5. Robot performs the action.
